# Exercise 06 - Business Day Check

## Objective

Read a weekday name in lowercase letters and check whether it is a business day.

## Concepts Practiced

- User input with `Scanner`
- `String` values
- `equals` for string comparison
- `if` and `else`
- Logical OR operator (`||`)

## How It Works

The program reads a day name and compares it with the lowercase names from Monday to Friday. If any comparison is true, it prints that the day is a business day. Any other value goes to the `else` branch.

## Example

```text
Input:
monday

Output:
Enter the day of the week (in lowercase letters):
monday is a business day.
```
