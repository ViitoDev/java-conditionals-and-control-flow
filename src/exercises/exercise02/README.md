# Exercise 02 - Student Grade Status

## Objective

Calculate the average of two fixed grades and print the student's status based on that average.

## Concepts Practiced

- `double` variables
- Arithmetic average calculation
- `if`, `else if`, and `else`
- Greater-than-or-equal comparison (`>=`)
- Multi-branch decision-making

## How It Works

The program stores two grade values, calculates their average, and checks the result against two limits. Averages of `7` or higher are approved, averages of `5` or higher go to remedial classes, and lower averages are rejected by the final `else` branch.

## Example

```text
Input:
No user input

Output:
The student have 5- media.
Repproved!
```
