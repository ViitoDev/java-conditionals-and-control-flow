# Exercise 05 - Purchase Discount

## Objective

Read a purchase amount and apply a 10% discount when the amount is at least `100`.

## Concepts Practiced

- User input with `Scanner`
- `double` values
- `if` and `else`
- Greater-than-or-equal comparison (`>=`)
- Multiplication and subtraction
- Conditional price calculation

## How It Works

The program reads the purchase amount. If the value is `100` or more, it calculates 10% of the amount, subtracts that discount, and prints the final price. Otherwise, it prints the original amount without applying a discount.

## Example

```text
Input:
120

Output:
Enter the purchase amount:
Discount applied!
Final value is : 108.0
```
