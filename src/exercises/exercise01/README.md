# Exercise 01 - Even or Odd

## Objective

Read an integer from the user and determine whether it is even or odd.

## Concepts Practiced

- User input with `Scanner`
- `if` and `else`
- Modulo operator (`%`)
- Equality comparison (`==`)
- Integer variables

## How It Works

The program asks the user for a number. It uses the remainder of the division by `2` to decide if the number is even. If the remainder is `0`, it prints that the number is even; otherwise, it prints that the number is odd.

## Example

```text
Input:
4

Output:
Enter a number:
The number is even
```
