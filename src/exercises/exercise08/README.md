# Exercise 08 - Donor Compatibility Check

## Objective

Read a donor's age and weight and print the donor compatibility message.

## Concepts Practiced

- User input with `Scanner`
- Integer and `double` values
- `if` and `else`
- Less-than comparison (`<`)
- Greater-than comparison (`>`)
- Logical AND operator (`&&`)
- Logical OR operator (`||`)

## How It Works

The program reads age and weight. It checks whether the donor has weight below `55` and age below `18`, or whether the age is greater than `65`. In the current code, both the `if` branch and the `else` branch print the same message.

## Example

```text
Input:
30
70

Output:
Enter your age:
Enter your weight:
The donor is not compatible.
```
