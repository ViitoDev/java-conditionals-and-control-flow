# Exercise 07 - Loan Range Check

## Objective

Read a loan amount and check whether it is within the permitted range from `1000` to `5000`.

## Concepts Practiced

- User input with `Scanner`
- `double` values
- `if` and `else`
- Less-than comparison (`<`)
- Greater-than comparison (`>`)
- Logical OR operator (`||`)

## How It Works

The program reads a loan amount. If the amount is below `1000` or above `5000`, it prints that the value is outside the permitted range. Otherwise, it prints that the value is within the permitted range.

## Example

```text
Input:
2500

Output:
Enter the Loan amount :
The value 2500.0 is within the permitted range.
```
