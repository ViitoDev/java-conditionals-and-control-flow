# Exercise 03 - Password Check

## Objective

Read a password attempt from the user and compare it with a fixed numeric password.

## Concepts Practiced

- User input with `Scanner`
- Integer variables
- `if` and `else`
- Equality comparison (`==`)
- Simple authentication logic

## How It Works

The program stores `123456` as the correct password. It reads a password attempt from the user and compares both integer values. If they match, it prints a success message; otherwise, it prints a failure message.

## Example

```text
Input:
123456

Output:
Enter your password:
Password Correct!
```
