# Exercise 04 - Number Comparison

## Objective

Read two numbers and print whether the first number is greater than, less than, or equal to the second number.

## Concepts Practiced

- User input with `Scanner`
- `double` values
- `if`, `else if`, and `else`
- Greater-than comparison (`>`)
- Less-than comparison (`<`)
- Equality handled by the final `else`

## How It Works

The program asks for two decimal numbers. It first checks if the first number is greater than the second, then checks if it is less than the second. If neither condition is true, the numbers are treated as equal.

## Example

```text
Input:
10
5

Output:
Enter the number 1 :
Enter the number 2 :
The number 1 is greater than to than the number 2
```
